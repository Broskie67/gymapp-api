`gymAppBackend` is a TypeScript and Express backend for a gym application, designed to provide secure authentication, user profile management, and a reliable API foundation for fitness-related features. The project uses JWT-based authentication, SQL Server for persistent data storage, and structured middleware for validation, authorization, and error handling.

## Features

- User registration and login
- JWT access token authentication
- Refresh token and logout flow
- Protected user routes
- User profile retrieval and updates
- Password update functionality
- Database health check endpoint
- Automated testing with Vitest and Supertest

## Tech Stack

- TypeScript
- Node.js
- Express
- SQL Server
- JSON Web Tokens (JWT)
- Vitest
- Supertest

## Purpose

This backend is built to serve as the core API for a gym or fitness application, handling user account security and profile operations while providing a scalable starting point for future features such as workouts, memberships, progress tracking, and scheduling.
